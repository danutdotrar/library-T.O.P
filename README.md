This project was an assignment from "The Odin Project".

A book library in which you can add the books that you'd read/want to read and delete which one you want.

Learned more about:
- Objects;
- Object Constructors;
- Prototypes and Prototypal Inheritance
- JavaScript;
- HTML;
- CSS.
